## Production Notes

- **Tone:** Confident, visionary, technical but accessible. Not hype-y. Infrastructure voice, not trader voice.
- **Pacing:** Fast but not rushed. Each screen should be visible for 3-5 seconds minimum.
- **Music:** Subtle electronic/ambient — terminal aesthetic vibes. Low volume under voiceover.
- **Screen recording:** Use bobbyprotocol.xyz in dark mode. Chrome with clean toolbar.
- **Resolution:** 1920x1080 minimum. Record at 60fps if possible.
- **Key moments to nail:**
  1. The Five Flywheels section (proves defensibility thinking)
  2. The Harness Console with events + memory tabs (proves auditability)
  3. The guardrails section with fail-closed manifesto (proves safety)
  4. The live tx feed scrolling (proves it's real)
  5. The Three Outcomes section (proves deterministic control)
  6. The trust score + bounties (proves accountability)
